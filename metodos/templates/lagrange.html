Lagrange

  • Situaciones recomendadas: Funciona bien cuando los puntos de datos están distribuidos uniformemente.
  • Ventaja: Permite la interpolación de cualquier conjunto de puntos dados sin necesidad de recalcular todo el polinomio al agregar nuevos puntos..
  • Desventaja: Puede ser ineficiente y numericamente inestable para conjuntos de datos grandes, lo que puede resultar en errores significativos.
Ten en cuenta a la hora de ingresar tus datos

El método de interpolacion de Lagrange permite calcular el polinomio interpolante con el conjunto de Vectores 'x' y 'y'

Los valores de 'x' y 'y' deben estar escritos asi [ a;b;c;d ], el tamaño de ambos vectores debe ser el MISMO

Solo se pueden ingresar valores numericos
