Vandermonde

El método de interpolacion de Vandermonde

  • Convergencia: Exacta para un conjunto dado de puntos, pero puede ser inestable para un gran número de puntos.
  • Situaciones recomendadas: Cuando se necesita un polinomio de interpolación que pase exactamente por todos los puntos dados.

Casos de uso:

Ten en cuenta a la hora de ingresar tus datos

El método de interpolación de Vandermonde utiliza una matriz para encontrar un polinomio que pase exactamente por un conjunto de puntos dados. El proceso implica construir la matriz de Vandermonde a partir de los puntos \(x\) y resolver un sistema lineal para obtener los coeficientes del polinomio.

A tener en cuenta al ingresar datos

Los valores de 'x' y 'y' deben estar escritos asi [ a b c d ], el tamaño de estos debe ser el mismo

Los valores de 'x' y 'y' pueden ser iguales pero no identicos, los valores de cada conjunto deben ser distintos entre si

Los valores de 'x' y 'y' deben ser numericos
